# Custom Crypto Icons

Upload hier je eigen SVG of PNG bestanden voor crypto munten.

## Bestandsnamen

Gebruik exact deze namen (hoofdlettergevoelig):

- `DSHP.svg` of `DSHP.png` - DigiSheep
- `ORX.svg` of `ORX.png` - Orex  
- `LNTR.svg` or `LNTR.png` - Lentra
- `SIL.svg` or `SIL.png` - Silica
- `REX.svg` or `REX.png` - Rex
- `ORLO.svg` or `ORLO.png` - Orlo

## Hoe te gebruiken

1. **Upload je SVG/PNG** naar deze folder (`/public/crypto-icons/`)
2. **Gebruik exacte bestandsnaam** (bijv. `DSHP.svg`)
3. **Refresh de app** - de custom icon wordt automatisch gebruikt

## Prioriteit

Het systeem checkt in deze volgorde:
1. Custom SVG in `/crypto-icons/` (bijv. `DSHP.svg`)
2. Custom PNG in `/crypto-icons/` (bijv. `DSHP.png`)
3. Standaard image in `/public/` (bijv. `dsheep.png`)
4. Fallback emoji

## Aanbevolen formaat

- **SVG**: Beste kwaliteit, schaalt perfect
- **PNG**: Transparante achtergrond, minimaal 256x256px
- **Bestandsgrootte**: < 100KB per icon

## Voorbeeld

Als je een custom DigiSheep icon wilt:
1. Maak/download een SVG bestand
2. Hernoem naar `DSHP.svg`
3. Upload naar `/public/crypto-icons/DSHP.svg`
4. Refresh de app - klaar!
